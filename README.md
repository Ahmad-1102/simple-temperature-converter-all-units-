🌡️ Temperature Converter

A web-based temperature converter supporting Celsius, Fahrenheit, Kelvin, Rankine, and Réaumur.

Features

-Convert between 5 temperature units

-Input validation with error messages

-Handles same-unit conversions

-Results formatted to 2 decimal places

-Clean blue-themed UI


Supported Units

C - Celsius

F - Fahrenheit

K - Kelvin

R - Rankine

Re - Réaumur


Installation

open the link https://ahmad-1102.github.io/simple-temperature-converter-all-units-/temperature-converter/index.html


Usage

-Enter temperature value

-Type unit to convert FROM (C, F, K, R, Re)

-Type unit to convert TO

-Click Convert

-Result displays with unit symbol


Technologies

-HTML5

-CSS3 (Flexbox layout)

-Vanilla JavaScript
